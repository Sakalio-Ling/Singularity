## Server: Paper

Paper is a high-performance Minecraft server fork of Spigot. Benefits:
- Better performance than vanilla
- Plugin API (Bukkit/Spigot/Paper API)
- Active development
- Async chunk loading
- Entity activation range optimization

## Compatibility Notes

- Some vanilla mechanics may differ slightly from Paper
- Plugin list should be minimal to avoid interference with bot
- Keep server on default settings for reproducibility
- Use flat or specific seeds for benchmark experiments
