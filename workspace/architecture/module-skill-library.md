# Module: Skill Library
> Status: Design (M0)
## Purpose
Store, version, retrieve, and manage reusable action units.
## Skill Schema
name, description, parameters, preconditions, postconditions, required_items, failure_modes, implementation, examples, tests, version, success_rate, last_used, notes
## Layers
Primitive, Composite, Strategic, Social, Meta
## Interface
list_skills, get_skill, execute_skill, create_skill, update_skill, get_recommended_skills
