# P-005: DEPS (Describe, Explain, Plan and Select)

**Title**: DEPS: Describe, Explain, Plan and Select - Interactive Planning with LLMs for Open-World Multi-task Agents
**Year**: 2023
**Link**: https://arxiv.org/abs/2302.01560
**Authors**: Zihao Wang, Shaofei Cai, Anji Liu, et al.
**Priority**: P3

## Core Method
Interactive planning framework with LLM feedback loops:
1. **Describe**: Describe the current world state
2. **Explain**: Explain what actions are needed
3. **Plan**: Generate a plan
4. **Select**: Select the best action

## Key Contributions
- Iterative feedback loop improves multi-task completion
- Explicit separation of reasoning stages
- Shows LLMs can benefit from structured reasoning frameworks

## Relevance to Singularity
- **Planner module**: Our Planner already implements a similar think-act cycle
- **Reflection**: DEPS feedback loop maps to our Reflector module
- **Action Selection**: The Select stage informs our action controller design

## Key Takeaway
Structured reasoning stages (describe-explain-plan-select) outperform single-shot planning for multi-task scenarios. We incorporate this by having separate observe, think, reflect phases.
